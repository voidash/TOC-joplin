"Joplin is an excellent open source note taking application with plenty of features". 

## TOC-Joplin

TOC Joplin is a simple plugin that generates table of content based on Markdown headers. 
```markdown
# level 1 header
## level 2 header
### level 3 header 
#### level 4 header

Rendered as:

level 1
    level 2
        level 3 
            level 4
```
